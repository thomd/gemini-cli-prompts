# Gemini CLI Custom Prompts

Custom prompts for web development and devops engineering to be used in [Gemini CLI](https://geminicli.com/).

## Prompts

- `/code:architecture` Design a system architecture
- `/code:changelog` Add a new entry to the project's CHANGELOG.md file
- `/code:explain` Explain a technical concept
- `/code:review` Review the code changes on your current branch
- `/prompt:improve` Improve existing prompts
- `/prompt:template` Create prompt template

## Install

```
gemini extensions install thomd/gemini-cli-prompts
```
